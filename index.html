Getting Started

Introduction

Thank you for purchasing SCANDI — creative and lightweight minimalistic landing page template! This theme is best suitable for small businesses that wouil like to reinforce sales with an easy to customize promotion website. It looks beautifully end elegant on all types of screens and devices including Windows, Mac, iOS, Android, etc. The SCANDI theme has a well commented and easily customizable source code with detailed documentation. We hope you will enjoy it!

Theme Requirements

To use SCANDI, you must be running WordPress 5 or higher, PHP 5.6 or higher, and MySQL 5 or higher. We have tested it with Mac, Windows, and Linux. Below is a list of items you should ensure your host can comply with.

  • Check to ensure that your web host has the minimum requirements to run WordPress.
  • Always make sure they are running the latest version of WordPress.
  • You can download the latest release of WordPress from the official WordPress website. If you need help installing WordPress, follow the instructions in WordPress Codex. Below are all the useful links for WordPress information:

    WordPress Codex – General info about WordPress and how to install on your server.

    First Steps With WordPress – General info about multiple topics about WordPress.

    FAQ New To WordPress – The most popular FAQ’s regarding WordPress.

  • Always create secure passwords for FTP and Database.
  • Make sure no additional plugins are activated except the required plugins by the theme.

Installation

The process of installing the theme to your website consists of two parts - Theme Installation and Plugin Installation. For the correct work of the website, please, complete both.

Theme Installation

After purchasing SCANDI, please download the package from ThemeForest. All you need to do is move your mouse over your login name at the right top corner – after you log in ThemeForest, and then click Downloads. Here you will find all the items that you have purchased. Hover to Download button next to Scandi theme, you will see these download options:

  • Installable WordPress file only which you can upload when installing the theme
  • All files & documentation (full zip folder). You will need to extract and locate the installable WordPress file to upload when installing the theme

Install theme via WordPress

Step 1: Navigate to Appearance >> Themes

Step 2: Click Add New, then click Upload Theme >> Choose File:

Step 3: Navigate to the .zip file on your computer, then click Install Now:

Step 4: When the installation complete, click Activate. You will be redirected to Themes page with Scandi activated:

Install theme via FTP

Step 1: Unzip zip file package – the zip package is inside Scandy files folder you have downloaded from ThemeForest Scandi.

Step 2: Connect to your FTP server and navigate to the folder where your WordPress is installed.

Step 3: Go to wp-content/themes/ and upload theme folder (the path will look like this: wp-content/themes/scandi

Step 4: Open WordPress admin panel (navigate through the web-browser to the folder where you have installed the WordPress e.g www.yourdomain.com/wp-admin/ ) and go to Appearance >> Themes on admin sidebar. Activate Scandi.

Plugin Installation

After activating Scandi, you will see this notice:

Click Begin installing plugins. You will be navigated to Install Required Plugins page.

Simply check all of them (or all of the required plugins and some recommended plugins you like) and from the drop-down above the list select Install, then hit Apply.

When finishing, it should look like this:

Menu

For easy navigation through the page SCANDI theme has a responsive slide-in navigation menu available on click on the menu button in the top right corner of the page. By default, the menu is "empty". Wordpress allows you to configure the menu as you like.

How To Create A Page Menu

Step 1: Navigate to Appearance >> Menus in your admin sidebar.

Step 2: In the top right corner hit Screen Options. It will open a tab with boxes for possible menu item types.

Step 3: Uncheck all the boxes except for Custom Links and Title Attribute.

Step 4: Give your menu a name, then hit Create Menu.

Step 5: The theme uses only anchor links in the menu to navigate through the page. In the tab Custom Links enter the section URL and the text that will appear on the menu. Here is a list of section URLs:

  • Main screen - #home
  • About Us - #about
  • FAQ - #faq
  • Testimonials - #testimonials
  • Gallery - #gallery
  • Contacts - #contacts
Copy and paste the URL from this list. Make sure to include #. For the correct work of scroll the menu items must be in the same order as the corresponding sections of the page!

Step 6: To complete the setting of the menu item you've just created, open it by clicking on the arrow next to the menu item as shown in the picture. For the correct work of the menu, please fill in the Title Attribute field. This attribute must be the same is the URL but without the # sign. You can see the example in the picture.

Step 7: Make sure to check the Main Menu box below the menu items list as seen in the picture above.

Step 8: Don't forget to hit Save Menu every time you create and edit menu items.

How To Edit The Menu

Step 1: Go to Appearance >> Menus in your admin sidebar.

Step 2: You can rename the menu as you like. If you want to add more menu items, choose them from the left and hit Add to menu. You can also change and remove the items.

Step 3: Don't forget to hit Save Menu every time you create and edit menu items.

Custom Posts

In Scandi theme Testimonials, Gallery Images and FAQs are added as the Wordpress custom posts. To add, edit or delete such posts, navigate to the corresponding sections on your Wordpress admin panel.

Testimonials

How to Add Testimonials

Step 1: Go to Testimonials >> Add new in your admin sidebar.

Step 2: Fill in all the fields and set the Featured Image.

Step 3: Click Publish when you finish. The testimonial will now appear on the page. Make sure the "Display Section" is checked for the Testimonials section. If you don't see the Testimonials section on the page, go to Appearance >> Customize >> Testimonials and check the "Display Section" box.

How to edit testimonials

Step 1: To see the list of testimonials you added, edit or delete them, navigate to Testimonials >> Testimonials in your admin sidebar.

Step 2: Hover your mouse over the item you need and click on Edit to change the contents of the post or Trash to delete it permanently.

Gallery Images

How to Add Gallery Images

Step 1: Go to Gallery Images >> Add new in your admin sidebar.

Step 2: Enter a title for the new image and set the Featured Image.

Step 3: Click Publish when you finish. The gallery image will now appear on the page. Make sure the "Display Section" is checked for the Gallery section. If you don't see the Gallery section on the page, go to Appearance >> Customize >> Gallery and check the "Display Section" box.

How to edit gallery images

Step 1: To see the list of gallery images you added, edit or delete them, navigate to Gallery Images >> Gallery Images in your admin sidebar.

Step 2: Hover your mouse over the item you need and click on Edit to change the contents of the post or Trash to delete it permanently.

How to delete an image from the Media Library

Step 1: If you need to delete an image from the Media Library, go to Media >> Library in your admin sidebar.

Step 2: Click on the image you want to delete. In the opened modal window hit Delete Permanently.

FAQs

How to Add FAQs

Step 1: Go to FAQ Items >> Add new in your admin sidebar.

Step 2: Fill in all the fields and click Publish when you finish.

Step 3: Make sure the "Display Section" is checked for the FAQ section. If you don't see the FAQ section on the page, go to Appearance >> Customize >> FAQ and check the "Display Section" box.

How to edit FAQs

Step 1: To see the list of testimonials you added, edit or delete them, navigate to FAQ Items >> FAQ Items in your admin sidebar.

Step 2: Hover your mouse over the item you need and click on Edit to change the contents of the post or Trash to delete it permanently.

See below for a screenshot of our Customize panel.

How To Display and hide sections

Each section has a display option. By default, this option is set to false for all the sections except for the main screen section, the header and the footer. In order to make a section visible on your page, follow the steps below:

Step 1: Navigate to Appearance >> Customize. On the customization panel choose the section you'd like to display.

Step 2: Check the ‘Display section’ box. You can now see the section on the page. To hide a section, uncheck the box. Don’t forget to click Publish after you have changed the ‘Display section’ option.

How To Change theme colors

Step 1: Go to Appearance >> Customize >> Color Scheme on your Wordpress Admin Panel.

Step 2: On the panel you will see two color pickers - one for the primary color, which is the color of the background elements, and one for the secondary color, which is the color of the buttons and list bullet points. Click on the one you would like to change and choose any color on the color picker, or paste in the HEX value of the color you want. You can always switch the colors back to default theme colors by clicking Default button.

Step 3: Click Publish to save the changes.

How to change logo

Step 1: Go to Appearance >> Customize >> Header on your Wordpress Admin Panel.

Step 2: Click on Change Image button below the logo image. It will open the Media Library.

If your logo image is not in the library yet, hit Upload Files and select the necessary image. Once it’s uploaded, add Alt attribute to the image and hit Choose Image button.

Step 3: Click Publish to save the changes.

How to add video

Step 1: Go to Appearance >> Customize >> Banner with video on your Wordpress Admin Panel.

Step 2: Make sure the “Display section” box is checked.

Step 3: The section uses YouTube player so your video must be uploaded on youtube. Once you upload your video, you get a message of successful upload and the link to the video. You only need the part of the link that contains the unique idetifier of the video - a set of numbers and letters that goes right after ‘youtu.be/':

Copy this part only and paste it into the Video URL option.

Step 3: Click Publish to save the changes.

How to customize theme typography

Step 1: Go to Appearance >> Customize >> Typography >> Default Typography on your Wordpress Admin Panel. If you cannot see the Typography tab on the customize panel, go back to Plugin Installation and install the required Easy Google Fonts Plugin.

Step 2: Choose the kind of the typographical elements you'd like to customize - paragraph text or headings - and click Edit Font. You will see a set of different options for customization.

In the Styles tab, you can change the Font Family, Font Weight/Style, Text Decoration and Text Transform. You can always switch back to the default style by picking the Theme Default option.

In the Appearance tab, you can change the Font Color, text Background Color, Line Height and Letter Spacing. You can always switch back to the default style by clicking Reset.

In the Positioning tab, you can change the Margin, Padding, Border and Border Radius of the text block, and Display options. You can always switch back to the default style by picking the Theme Default option.

Step 3: Click Publish to save the changes.
